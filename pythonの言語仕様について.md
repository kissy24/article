## Pythonとはどういう言語か

### 1. 記述の自由度の高さ
- 「手続き」でも「オブジェクト指向」でも「関数型」でもコーディングできる。
- 例外は投げても、返しても良い
```py
# example
def hoge(hoge: int):
    # print(hoge(1)) は error を返す
    if hoge == 1:
        return Exception("error")
    # hoge(2) は Exception: error を投げる
    if hoge == 2:
        raise Exception("error")
```
- 型付けしてもしなくても良い
  - `typing`モジュールを利用することによる型の指定ができる

- 大体なんでも`return`できる
  - `class`オブジェクトとかも返せる

### 2. インタプリタ言語
- 機械語に逐次翻訳される
- インタプリタであるがJITとかコンパイルとか抜け道が豊富

### 3. 少ないコード行数
- if文のelse省略
  - このおかげでネストしなくてもよくなったりする恩恵もある
- リスト内包表記
```py
# for文を使ったlistへの追加
hoge = []
for i in range(10):
    hoge.append(i)

# リスト内包表記
hoge = [i for i in range(10)]
```
- ラムダ式
- セイウチ演算子
- map関数
